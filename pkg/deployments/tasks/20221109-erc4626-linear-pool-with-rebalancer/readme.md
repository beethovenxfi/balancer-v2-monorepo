# 2022-11-09 - ERC4626 Linear Pool With Rebalancer

Third deployment of the `ERC4626LinearPoolFactory`, for Linear Pools with a ERC4626 yield-bearing token. This version adds the Rebalancer to the pool deployment, allowing for significantly easier management of linear pool main token balances.
